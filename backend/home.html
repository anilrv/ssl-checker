SSL Issue Checker — backend service

This is the backend service for the SSL Issue Checker Chrome extension. It performs a live TLS handshake against the site you're viewing and reports its certificate's organization, issuer, validity, and any certificate issues (expired, self-signed, hostname mismatch, incomplete chain, untrusted root, or outdated TLS protocol support).

This page isn't meant to be used directly — install the extension to check sites as you browse.
